offen:
$\boxed{\dot {Q} - \dot {W}= \dot {m} ((h_2 - h_1) +\frac{{w_2}^2-{w_1}^2}{2}+g(z_2-z_1))}$

Wirkungsgrad: $COP = Q/W$ $\eta=W/Q$

Bemerkung: Enthalpie statt innerer Energie wg. Strömungsarbeit,
kinetische und potentielle Energie kann man meistens weglassen

adiabatisch: $\dot{Q}=0$ es bleibt
$-\dot {W}= \dot {m} ((h_2 - h_1)$
